# 🎬 Netflix Content Analysis Dashboard

📊 An interactive **Power BI Dashboard** built using the **Netflix Titles Dataset** to analyze content trends, genres, ratings, directors, and countries.

This dashboard provides meaningful business insights through interactive visualizations and KPIs using **Power BI**, **Power Query**, and **DAX**.

# 📂 Dataset Information

The dashboard is built using the **Netflix Titles Dataset**, which contains information about:

- Movies & TV Shows
- Directors
- Cast
- Countries
- Genres
- Ratings
- Release Year
- Date Added
- Duration

---

# 💡 Key Insights

- Movies significantly outnumber TV Shows.
- Content releases increased rapidly after **2015**.
- **International Movies** and **Dramas** are among the most popular genres.
- **TV-MA** is the most common content rating.
- A few countries contribute the majority of Netflix content.
